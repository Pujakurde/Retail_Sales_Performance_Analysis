## 🚀 Project Overview

This project presents a fully interactive multi-page Sales Analytics Dashboard built using Power BI.
The objective was to transform structured retail sales data into actionable business insights through clean visualization and storytelling.

---

The dashboard is organized into four analytical layers:

### 1️⃣ Executive Revenue Overview

* Total Revenue, Orders, Customers, AOV (KPI Cards)
* Monthly Revenue Trend
* Dynamic slicers (Year, Region, Segment, Category)
